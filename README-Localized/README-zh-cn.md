# Microsoft Graph：生成 Microsoft Graph 应用程序

此模块演练如何使用多种技术生成应用程序。

## 实验

在此实验中，你将使用 Graph SDK 和 Microsoft 身份验证库 (MSAL) 演练采用 Microsoft Graph 生成不同类型的应用程序。

- [实验手册](./Lab.md)
